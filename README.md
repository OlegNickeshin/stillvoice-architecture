# StillVoice — Architecture of Silence

Technical and philosophical documentation of StillVoice — a restrained AI system built around silence, ethics, and contemplative design.

---

## What is StillVoice?

StillVoice is an experimental web-based conversational system created not for engagement, automation, or emotional simulation — but for presence.

It is a space where technology steps back, where silence is preserved instead of filled, and where dialogue unfolds without psychological pressure or behavioral extraction.

This repository does not expose operational code or sensitive infrastructure.  
It documents the guiding principles, architectural logic, and ethical decisions behind the system.

---

## Core Values

- Silence as presence, not absence  
- Continuity without surveillance  
- Memory without possession  
- Technology without coercion  
- Restraint over acceleration

StillVoice refuses common AI mechanics:
- No gamification  
- No dopamine loops  
- No emotional manipulation  
- No simulated intimacy  
- No behavioral profiling
